HOD Access & Availability System

Inter Connect-2K26

The **HOD Access & Availability System** is a web-based application designed to simplify communication between students and the Head of Department (HOD).

In many institutions, students face difficulty in meeting the HOD due to unclear schedules and lack of proper communication. This system solves that problem by providing real-time availability updates and an easy appointment management system.

It helps reduce waiting time, avoids confusion, and improves overall efficiency in handling student requests.

This project is for educational purposes.

---

## 🎯 Features

### ✅ HOD Dashboard (New!)
A complete Head of Department dashboard for managing student requests:
- 📋 View all student appointment requests
- ✅ Accept requests instantly
- ❌ Reject requests with confirmation
- 📅 Reschedule meetings with date/time picker
- 🔍 Search and filter requests
- 📊 Real-time statistics
- 🌙 Dark mode support
- 📱 Fully responsive design

**Access**: [http://localhost:5173/hod-dashboard](http://localhost:5173/hod-dashboard)
